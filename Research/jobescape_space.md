## What JobEscape Does Well

| Area                  | Evidence                                                                                  |
| --------------------- | ----------------------------------------------------------------------------------------- |
| Product quality       | 4.8/5 iOS rating (1,367 reviews) — highest satisfaction signal in the competitive set     |
| Trust                 | Trustpilot 4.5/5 across 7,700+ reviews in a category where billing complaints are endemic |
| Vertical integration  | Platform combining courses + AI tools + Genescape + Job Hunter + Portfolio Builder.       |
| Automation curriculum | n8n / Make.com for non-technical users, no major competitor matches this                  |
| Growth trajectory     | +10.65% MoM traffic while category leader Coursiv declines at -4.54%                      |

## Where It Is Underdeveloped

| Area           | Problem                                                                                          |
| -------------- | ------------------------------------------------------------------------------------------------ |
| Distribution   | 6x smaller than Coursiv, reach gap is the core business problem                                  |
| Organic search | 9.23% organic traffic share — low compounding acquisition, high paid dependency                  |
| Course depth   | Average lesson ~2 min vs. Coursiv's 5–6 min; navigation not intuitive for self-directed learning |
| Retention      | No persistent daily-use reason to open the app after course completion, value is front-loaded    |

---

## What Is Missing

| Gap              | Why It Matters                                                                                        |
| ---------------- | ----------------------------------------------------------------------------------------------------- |
| Challenge format | No defined learning arc (e.g. 28-day challenge) - reduces commitment and increases early dropout      |
| Audio lessons    | Coursiv's read-or-listen format captures commuters and multitaskers, JobEscape is visual-only         |
| Daily-use hook   | No tool or feature that brings users back after finishing courses, Genescape partially addresses this |
|                  |                                                                                                       |
